# TimeTrace

一个记录重要日期的倒数 / 累计日 App，从 Android 迁移至鸿蒙（HarmonyOS）。

## 功能

- 创建倒数日 / 累计日事件
- 首页卡片列表，支持滑动删除
- 自定义背景图片 + 遮罩强度
- 置顶展示重要事件
- 全屏预览 / 保存为纯净长图
- 深色模式 / 浅色模式 / 跟随系统
- 数据备份与恢复
- 多语言支持
